# DeadlineKu
DeadlineKu adalah aplikasi manajemen tugas mahasiswa yang dibuat untuk membantu pengguna mengatur tugas kuliah dan deadline secara lebih terstruktur.

Melalui aplikasi ini, mahasiswa dapat:
- menambahkan tugas baru
- menentukan tanggal deadline
- melihat daftar tugas yang belum selesai
- menandai tugas yang sudah dikerjakan dengan checklist
- mendapatkan pengingat sebelum deadline tiba

Aplikasi ini cocok digunakan oleh mahasiswa yang memiliki banyak tugas, kesulitan mengatur waktu, atau sering lupa mencatat deadline.

Tujuan utama DeadlineKu adalah membantu mahasiswa menjadi lebih terorganisir, lebih tepat waktu dalam mengumpulkan tugas, dan mengurangi risiko tugas terlupakan.

## Teknologi
- React
- TypeScript
- Tailwind CSS
- Supabase
- Vite
